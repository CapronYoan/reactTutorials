Il s'agit d'une formation react permettant d'en comprendre les bases afin de bien débuter sur cette techno.

Principalement, les fichiers à voir et à éditer sont numérotées et se trouvent dans le dossier js. Cependant, dans le fichier index.html, il vous faudra commenter l'appel de chaque fichier javascript terminé et décommenter le suivant à mesure que vous avancé dans la formation.

Lancez préalablement un serveur http type apache ou autre, contenant les fichiers et dossiers de ce répertoire. 

En complément voici le lien vers la documentation officielle réact en Français: https://fr.reactjs.org/docs/getting-started.html
